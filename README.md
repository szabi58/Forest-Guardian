# Forest Guardian

A 3D forest game built with React Three Fiber. Run and develop locally in Cursor (or any editor).

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. (Optional) For NPC dialogue AI, set `GEMINI_API_KEY` in `.env.local` to your [Google AI / Gemini](https://ai.google.dev/) API key.
3. Start the app:
   ```bash
   npm run dev
   ```
